Thomas Heim // UX UI Designer

Racket Rampage

My work in Racket Rampage, Joyride Games' flagship project, was to update the current interface designs for existing feature, as well as to design some new interfaces from the ground up, in order to give it a new style and look to the game. For this i worked through conceptualization, wireframing and visual design before handing off the work to the engineering team for implementation, providing support when necessary. Design work also included mapping out the architecture and flow for each of the updated layouts and features

Other aspects of my work included structuring the first time user experience flows for the general induction into the game as well as for specific features that unlocked later on, as well as helping with the sequencing of the animations of each ui flow

Overall this was a project where i could deepen my understanding of mobile game design, as well as make use of my self-sufficiency to take care of multiple tasks throughout the game in order to provide support both to my lead as well as the different teams involved on each feature
